## 필수 개념
1. 외생변수
	- 모델링하지 않는 변수 (해당 정의는 최근에는 지양)
	- 회귀분석에서 오차항에 가까운 설명 (관측되지 않은 모든 요인을 포괄하는 항)
	- 관측되지 않은 (시스템 내에 포함되지 않은) 어떠한 다른 변수에도 영향을 받지 않는 변수

2. 내생변수
	- 관측되지 않은 (시스템 내에 포함되지 않은) 요인에 의해 영향을 받는 변수
	- 예시 : 가격할인여부에 따른 주간 판매량에서 할인여부와 비즈니스사이즈는 내생변수

3. 공변량
	- 독립변수 외에 종속변수에 영향을 미칠 수 있어 통계적으로 통제해야하는 외생변수
	- 역할 : 통제변수(주 관심 효과 왜곡하지 않도록 보정), 잡음제거(종속변수에 영향을 주는 혼란 변수 효과 제거)



4. ATE
	- 평균처치효과 (average treatment effect)
5. ATT
	- 실험군에 대한 평균 처치효과 (average treatment effect on the treated)

6. CATE
	- 조건부 평균 처치효과
	- 변수 X로 정의된 그룹에서의 처치효과로 개인화에 유용
	- 예 : 이메일이 45세이상 고객, 젊은 고객에 미치는 영향을 파악

7. 편향 (Bias)
	- 정의 : 인과관계와 연관관계를 다르게 만드는 요소
	- 평균 처치효과에 대한 추정량이 실제 ATE를 시스템적으로 과소 또는 과대평가하는 경우, 편향되었다고 함
	- 수식 : Bias = E[β̂ - β] (β̂은 추정량, β는 추정하려는 모수)
	- 연관관계의 구조 : 연관관계 = ATT + 편향
		- 예 : 할인회사와 그렇지 않은 회사의 평균 판매량 차이 = 처치효과 + 회사규모 등의 편향
	- 편향의 원인 : 교란(경영방식, 위치, 할인 시기 등)
	- 해결 조건 : 가격할인으로 판매량이 얼마나 증가하는지 결정하려면, 할인하는 회사와 하지 않는 회사가 평균적으로 서로 유사성이 있어야 함
		- 즉, 실험군과 대조군이 "교환가능(exchangeable)"해야 함

8. 인과적 식별 (Causal Identification)
	- 편향을 없애는 과정
	- 인과효과를 정확하게 추정하기 위해 교란요인을 제거하거나 통제하는 방법론적 절차

9. 독립성 가정 (Independence Assumption)
	- 처치가 잠재적 결과에 어떤 정보도 제공하지 않음을 의미
	- 즉, 처치여부에 관계없이 실험군과 대조군 모두 서로 비교가능하고 구별할 수 없다는 뜻
	- 이 가정이 충족되면 편향이 제거되고 인과효과를 식별할 수 있음
